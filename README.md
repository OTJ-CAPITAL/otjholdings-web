# otjholdings.com

Website for **OTJ Holdings** — the parent company behind OTJ Capital, OTJ App, and OTJ Fellowship.

---

## What this site is

The top-level company page. Explains the mission, links to all three operating entities, and communicates the broader thesis: building Africa's financial infrastructure from Nairobi.

**Sections:**
- Hero — parent company positioning with links to all three entities
- Mission — why Africa's capital markets are the final frontier
- Portfolio — OTJ Capital, OTJ App, OTJ Fellowship with descriptions

---

## Portfolio companies

| Company | Domain | Description |
|---|---|---|

## Stack

| Layer | Technology |
|---|---|

---

## Development

```bash
npm install
npm run dev
# → http://localhost:3000
```

---

## Deploy

1. Connect `OTJ-CAPITAL/otjholdings-web` to Vercel
2. No environment variables required
3. Set custom domain: `otjholdings.com`
4. Every push to `main` auto-deploys
